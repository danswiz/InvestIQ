Nicholas Carlini from Anthropic's Safeguards team details an experimental approach called "agent teams" where multiple Claude instances work in parallel on a shared codebase without human intervention. As a stress test, 16 agents were tasked with building a Rust-based C compiler from scratch capable of compiling the Linux kernel.

Key Results:

  • ~2,000 Claude Code sessions over the project lifetime
  • $20,000 in API costs
  • 100,000-line compiler produced
  • Successfully compiles Linux 6.9 on x86, ARM, and RISC-V architectures

Core Innovation: A harness that keeps Claude in a continuous loop‚Äîwhen it finishes one task, it immediately picks up the next without waiting for human input. This enables sustained, autonomous progress on complex engineering tasks.

The paper also covers lessons learned about designing test harnesses for long-running autonomous agents, structuring work for parallel progress, and understanding the limitations of this approach.
