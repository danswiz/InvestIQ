Summary

Large language model (LLM)-powered multi-agent systems (MAS) demonstrate remarkable collective intelligence, where multi-agent memory serves as a pivotal mechanism for continual adaptation. However, existing multi-agent memory designs remain constrained by two fundamental bottlenecks: (i) memory homogenization arising from the absence of role-aware customization, and (ii) information overload induced by excessively fine-grained memory entries.

To address these limitations, the authors propose LatentMem, a learnable multi-agent memory framework designed to customize agent-specific memories in a token-efficient manner.
