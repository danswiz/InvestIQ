## Voice Call Setup Task
Dan wants to talk to me via voice call. Research findings:

### OpenClaw Voice-Call Plugin
- **Location:** `/opt/homebrew/lib/node_modules/openclaw/extensions/voice-call/`
- **Status:** Disabled (needs configuration)
- **Providers supported:** Twilio, Telnyx, Plivo, Mock

### What's needed:
1. **Twilio Account** (or Telnyx/Plivo)
   - Sign up at twilio.com
   - Get a phone number (~$1/month)
   - Get Account SID + Auth Token
   
2. **Public Webhook** for callbacks:
   - Option A: ngrok (free tier works)
   - Option B: Tailscale Funnel
   - Option C: Public URL

3. **Config to add to openclaw.json:**
```json
{
  "plugins": {
    "entries": {
      "voice-call": {
        "enabled": true,
        "config": {
          "provider": "twilio",
          "fromNumber": "+1TWILIO_NUMBER",
          "toNumber": "+1DAN_PHONE",
          "twilio": {
            "accountSid": "ACxxxxxxxx",
            "authToken": "TOKEN"
          },
          "tunnel": {
            "provider": "ngrok"
          }
        }
      }
    }
  }
}
```
